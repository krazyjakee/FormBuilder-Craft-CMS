FormBuilder is a Craft CMS plugin that lets you create forms for your front-end. Each form can have submission notification sent to form admin as well as submitter. All submission entries are saved to the database. You can easily view/update forms from the plugin's control panel. Form entries are also displayed on the plugin's entries page. You can view submissions and delete them if you want.

  • Installation

    • 1 Download the plugin from github git@github.com:roundhouse/FormBuilder-Craft-CMS.git

    2 Rename the folder to FormBuilder and place it into your Craft CMS Plugin directory.

    3 Navigate to your plugins page /settings/plugins and click Install

    4 New menu will appear in your toolbar called FormBuilder.

  • Create Fields

    • 1 Got to Settings->Fields and hit "+ New Field"

    2 Fill out the field's information. Make sure to use | FormBuilder | fields only

  • Create Forms

    • 1 Go back to FormBuilder forms page and hit "+ New Form"


    Form Settings

    Form Name - Enter name of your form

    Handle - Will be generated automatically

    Email Subject - This will be subject line for email notifications

    Use AJAX? - Check this if you want the form submission via ajax, this will use javascript to validate 'required' fields

    Redirect To Success Page - Check if you want to redirect to a success page (ex: "/thank-you")

    Redirect URL - Enter url to a success page

    Messages

    Success Message - Enter success message for form submissions

    Error Message - Enter error message for form submissions

    Notifications

    Send Notification? - Check this if you want to notify form owner of form submission

    To Email - Enter email where the submission notification will be sent to?

    Email Template Path - Path where the email template lives

    Notify Registrant? - Check this if you want to notify the form submitter of their successful submission. You need to first save the form for this to show up.

    Registrant Email Template Path - Path where the email template lives

    Notification Field - Path where the email template lives. You must first add some fields to the fields section and save the form for this to show up. Also this field requires "| FormBuilder | Email Address" fieldtype to be used.

    Fields

    Fieldsets - You can create many fieldsets by hitting "Add Fieldset"

    Create New Field - This will redirect you to settings->fields where you can create a new field

    Save! Now your form is ready for use. Take note of the form Handle, we will use that in the next steps to display our Subscription Form.

  • Required Fields

    • If you want to make required fields you can do that by clicking on the "cog" icon in the Fields section. See screenshot. This will add HTML5 "required" attribute to the input field. If your form uses Ajax to post submissions, javascript will validate those fields (using Parsleyjs).
